 Disability Support Management System

A web-based system designed to manage and streamline disability-related services such as registration, support requests, and assistance tracking. The system aims to improve accessibility, coordination, and service delivery between users, caregivers, and service providers.

 Features

- User registration and profile management
- Disability information recording
- Request and tracking of support services
- Admin dashboard for managing users and requests
- Role-based access (Admin, User, Service Provider)
- Accessible UI design considerations

 Tech Stack

- Frontend: HTML, CSS, JavaScript 
- Backend: PHP 
- Database: MySQL
- Server: XAMPP
